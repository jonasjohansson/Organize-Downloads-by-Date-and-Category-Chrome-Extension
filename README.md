# Organize Downloads by Date and Category

Automatically organizes your downloads into folders by category and date.

**Before:** `Downloads/photo.jpg`

**After:** `Downloads/2026/image/02/15/photo.jpg`

## Features

- 14 file categories with 150+ extensions supported
- Customizable folder structure via popup (click the extension icon)
- Works silently in the background

## Categories

| Folder | Extensions |
|---|---|
| image | jpg, png, gif, svg, webp, psd, raw, heic, avif... |
| video | mp4, mkv, avi, mov, webm, flv... |
| audio | mp3, wav, ogg, flac, aac, m4a... |
| apps | exe, dmg, msi, deb, pkg, apk... |
| text | pdf, doc, docx, txt, md, rtf, epub... |
| spreadsheet | xls, xlsx, csv, ods, numbers... |
| presentation | ppt, pptx, odp, key |
| archive | zip, rar, 7z, tar, gz, iso... |
| code | js, ts, py, java, c, cpp, go, rs... |
| web | html, css, json, xml, yaml, sql... |
| font | ttf, otf, woff, woff2... |
| model | obj, fbx, stl, blend, gltf... |
| torrent | torrent |
| system | dll, so, dylib, bin... |
| other | everything else |

## Folder Structure

Default: `YEAR/CATEGORY/MONTH/DAY/FILE`

Click the extension icon to customize. Available tokens: `{year}`, `{month}`, `{day}`, `{category}`

## Installation

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/organize-downloads-by-dat/bmjnpionejeiabjlfpjgejhdcabgpap)

## Manual Installation

1. [Download the extension](https://github.com/jonasjohansson/Organize-Downloads-by-Date-and-Category-Chrome-Extension/archive/master.zip)
2. Extract to a folder
3. Open Chrome → Extensions → Enable Developer Mode
4. Click "Load unpacked" and select the extracted folder
